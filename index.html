Quick Summary¶

TextFlow is a minimalist interface for full text processing with natural language tools. As of now, the app is capable of running projects from end to end. i.e. it is capable of getting annotations, processing them and creating models on the fly. There are two (2) main components of this system:

  1. Annotator

    1. Interface for Annotation

    2. Annotation Agreement

  2. Modeling

    1. Automatic Model Training

    2. Automated Model Evaluation

The platform is highly extensible for running a specific NLP project and very much beneficial for researchers.
